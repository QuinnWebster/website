Robot

For our first-year design project, my team and I build an autonomous robot capable of locating a target marked with an infrared sensor, navigating to it, and placing a ping pong ball on it.

This was an interesting project as it involved building the frame of the robot, wiring its motors and sensors, and programming the robot to locate and navigate to the target. All code was written in ROBOTC, a C-based language designed for VEX robots.

Technical Skills Developed:

  • Developed a custom algorithm for efficiently locating the IR target.
  • Integrated responsive code to ensure the robot continuously adjusted its path outowards the target
  • Our robot successfully completed the required task on the first attempt within the required time, earning an A for the project
