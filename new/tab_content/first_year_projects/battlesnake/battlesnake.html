BattleSnake

BattleSnake is a multiplayer programming game where participants build and program their own snake to compete against others on a grid. The goal is to outmaneuver opponents, strategically consume food, and avoid obstacles to become the last snake standing.

This was my first project using Python, having previously worked only with C and Java. I partipated in this project as part of a UVic hackathon, teaming up with three friends.

Our snake advanced to the the semi-finals of the competition, winning multiples rounds along the way. This was a great learning experience and a great introduction to Python.

Technical Skills Developed:

  • Python Programming: Developed proficiency in Python through real-time game development and problem-solving.
  • Game Logic and Mechanics: Designed and implemented core game mechanics of the BattleSnake game, including optimized movement, collision detection, and food consumption
  • Event-Driven Programming: Applied event-driven programming to dynamically update game state and enable autonomous decision-making during gameplay.
  • Colaboration and teamwork: Collaborated effectively with team members, dividing tasks and integrating contributions into a cohesive final project.
