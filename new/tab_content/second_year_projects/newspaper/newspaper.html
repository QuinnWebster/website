newspaper

This project involves webscraping top news websites from my hometown, Canmore AB, summarizing them, and sending me text messages with the summaries.

I first had the idea for this project because I felt disconnected from the ongoing events in Canmore. I wanted a quick and easy way to stay informed about what was happening in my hometown.

I used Python with the BeautifulSoup library to scrape the websites, and the Twilio API to send the text messages. This was a good introduction to web scraping, as I learned about the various HTML structures across different websites.

I used latent semantic analysis to summarize the articles, which was a new concept for me. It was interesting to see how a computer could understand the context of an article and generate a summary.

Technical Skills Developed

  • Using APIs
  • Text analysis
  • Text message automation
