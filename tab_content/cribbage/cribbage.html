This is a project that I developped to help teach my fundamental skills in web development. This was a fun project for me to make because I love playing cribbage and I wanted to make a tool that would help me improve my game.


Developed to enhance your cribbage playing experience, this project offers a user-friendly interface where players can input their hand's six cards and receive instant recommendations on which cards to keep and discard to maxami scoring potential.


Technical Skills Developped:

  • -Designed and coded a responsive web application using HTML, CSS, and JavaScript
  • -Implemented a custom algorithm to optimize cribbage hand evaluation
  • -Utilized JavaScript to handle user input and dynamically update the interface
  • -Applied CSS for styling and layout to enhance user experience
  • -Learned best practices in web development, including code organization and optimization
  • -Enhanced problem-solving skills through debugging and refining the application
