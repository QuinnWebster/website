This is a project that I developped with three friends to compete in a UVEC Hackathon. This was our first exposure to game development with Pygame


Developed to mimic air hockey game experience, this project allows users to play one on one game against their friends


Technical Skills Developped:

  • Implemented an interactive air hockey game using Python and Pygame
  • Utilized object-oriented programming (OOP) concepts to structure the game code efficiently
  • Developed game mechanics including player controls, puck movement, and collision detection
  • Applied Pygame's rendering capabilities to create visually appealing graphics and animations
  • Implemented game physics for realistic puck movement and collision dynamics
  • Integrated user input handling for player interaction and game control
